Face Detection with a Sliding Window

Click the image for more details!

The sliding window model is conceptually simple: independently classify all image patches as being object or non-object. This project serves as an introduction to methods used in implementing sliding window models. This project implements the simple sliding window detector of Dalal and Triggs. Dalal-Triggs focuses on representation more than learning and introduces the SIFT-like Histogram of Gradients (HoG) representation. My basic responsibility for this project was stringing together the detection pipeline and tuning the many parameters.

Camera Calibration and Fundamental Matrix Estimation with RANSAC

The goal of this project was to introduce camera and scene geometry. Specifically, camera projection matrix estimation, which maps 3D world coordinates to image coordinates, and fundamental matrix estimation, which relates points in one scene to epipolar lines in another.

Click the image for more details!

The goal of this project was to create a local feature matching algorithm using a simplified SIFT descriptor pipeline. I worked backward to develop a working pipeline, creating a working Feature Matching function, then a SIFT descriptor, and lastly a harris corner detector.
